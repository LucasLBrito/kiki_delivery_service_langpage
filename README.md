# Kiki Delivery Service — Langpage

Página estática (langpage) para o Kiki Delivery Service — site de apresentação/localização do serviço.

Demo: https://library-2tyr.vercel.app

## Descrição

Site estático desenvolvido em HTML/CSS/JavaScript para apresentar o Kiki Delivery Service. Esta versão funciona como página de idioma/landpage para o serviço, prontamente hospedada (ver demo acima).

## Tecnologias

- HTML
- CSS
- JavaScript (opcional)
- Hospedagem estática (Vercel, Netlify, GitHub Pages, etc.)

## Funcionalidades

- Layout responsivo para desktop e mobile
- Seções de apresentação do serviço, recursos e contato
- Fácil personalização de conteúdo e estilos

## Como executar localmente

1. Clone o repositório:
   git clone https://github.com/LucasLBrito/kiki_delivery_service_langpage.git
2. Entre na pasta do projeto:
   cd kiki_delivery_service_langpage
3. Abra o arquivo `index.html` no navegador ou execute um servidor local:
   - Com Python 3:
     python3 -m http.server 8000
     Acesse: http://localhost:8000
   - Com npm (serve):
     npm install -g serve
     serve .
4. Edite os arquivos em `index.html` e os recursos em `assets/` conforme necessário.

## Estrutura do projeto (exemplo)

- index.html — página principal
- assets/
  - css/ — arquivos de estilos
  - js/ — scripts JavaScript
  - img/ — imagens e ícones

(Ajuste conforme a estrutura real do repositório.)

## Deploy

- Deploy fácil em Vercel, Netlify ou GitHub Pages. Basta apontar para a pasta raiz do projeto (contendo `index.html`).
- No Vercel: conecte o repositório e selecione a branch `master` (ou a branch padrão).
